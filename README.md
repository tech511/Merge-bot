# Telegram Video Merge Bot

## Features
- Merge MP4 & MKV videos
- /start command with image + caption + buttons
- Owner-only /set_image command
- Multi-user safe (max 4 users)
- Owner unlimited
- Log channel support

## Deployment
- Set Railway / VPS Variables:
  - API_ID
  - API_HASH
  - BOT_TOKEN
  - OWNER_ID
  - LOG_CHANNEL
